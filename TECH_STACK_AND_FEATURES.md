# Winter Casa Resort & Cottages - Tech Stack & Features

## Tech Stack

### Frontend Framework
- **Next.js 16.0.0** - React framework with App Router
- **React 19.2.0** - UI library
- **TypeScript 5.0.0** - Type-safe development

### Styling & Design
- **Tailwind CSS 4.0.0** - Utility-first CSS framework
- **PostCSS 8.4.0** - CSS processing
- **Autoprefixer 10.4.0** - CSS vendor prefixing
- **Google Fonts (Cormorant Garamond)** - Custom typography for headings and body text

### Animation & Interactions
- **Framer Motion 11.0.0** - Advanced animations and motion effects
  - Scroll-based animations
  - Mouse tracking interactions
  - Page transitions
  - Hover effects

### UI Components & Libraries
- **Swiper 11.0.0** - Touch-enabled slider/carousel for testimonials
  - Autoplay functionality
  - Pagination controls
  - Responsive breakpoints

### Development Tools
- **ESLint 9.0.0** - Code linting
- **eslint-config-next** - Next.js-specific linting rules

### Build & Deployment
- **Next.js Build System** - Optimized production builds
- **Image Optimization** - Next.js Image component for optimized images

## Features

### Navigation & Layout
- **Fixed Glass-morphism Navbar**
  - Dynamic color adaptation based on scroll position
  - Smooth scroll-based shrinking effect
  - Backdrop blur with transparency
  - Responsive navigation menu
  - Smooth anchor link scrolling

### Hero Section
- **Full-screen Video Background**
  - Auto-playing, looping video
  - Forest gradient overlay
  - Animated fog effects
  - Mouse-tracking parallax effect on content
  - Animated text entrance
  - Call-to-action button linking to booking

### About Section
- **Two-column Layout**
  - Descriptive content about the resort
  - Image gallery grid with hover effects
  - Scroll-triggered animations
  - Background image with overlay

### Cottages/Rooms Section
- **Room Type Showcase**
  - Three room types: Classic Room, Deluxe Room, Private Cottage
  - Image cards with hover animations
  - Responsive grid layout (1/2/3 columns)
  - Detailed descriptions for each accommodation
  - Scroll-reveal animations

### Testimonials Section
- **Interactive Carousel**
  - Swiper slider with autoplay
  - Multiple customer testimonials
  - Star rating display
  - Responsive slide count (1/2/3 based on screen size)
  - Glass-morphism card design
  - Pagination indicators

### Booking System

  - Breakfast preference selection

### Contact Section
